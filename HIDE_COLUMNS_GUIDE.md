# Hide Unnecessary Columns

## 🎯 Goal
Simplify your board to show only essential feedback columns.

---

## ✅ Columns to Keep (4 Essential)

1. **Column name** (Name) - Item identifier
2. **Email** - Optional contact info
3. **Feedback** - Main feedback text
4. **Category** - Feedback type

---

## ❌ Columns to Hide (4 Optional)

1. **Person** - Not needed for anonymous feedback
2. **Status** - Optional (can keep if you want to track review status)
3. **Date** - Auto-populated, not essential to display
4. **File** - Not needed for text feedback

---

## 📝 How to Hide Columns (30 seconds)

### Method 1: Hide Individual Columns

1. **Click the column header** (e.g., "Person")
2. Click the **three dots (⋮)** menu
3. Select **"Hide column"**
4. Repeat for: Status, Date, File

### Method 2: Use Column Settings

1. Click the **three dots (⋮)** next to any column
2. Select **"Customize columns"**
3. **Toggle OFF** the columns you want to hide:
   - Person
   - Status  
   - Date
   - File
4. Click **"Apply"**

---

Clean and focused on feedback collection! ✨

---

## 💡 Optional: Keep Status Column

If you want to track feedback review progress, **keep the Status column**:

```
| Column name | Status | Email | Feedback | Category |
|-------------|--------|-------|----------|----------|
```

This lets you mark feedback as:
- Working on it (reviewing)
- Done (implemented)
- Stuck (needs discussion)
- On Hold (future consideration)
